# UniTask — 単位を守る、大学生のタスク管理アプリ

**🔗 アプリURL:** https://unitask-pro.vercel.app

---

## 開発背景

大学で留年した経験から、出席コードや課題提出の期限管理の重要性を痛感。
「あと1回出席していれば」「あの課題を出していれば」という後悔をなくすために、
大学生の単位取得を徹底サポートするWebアプリを開発した。

---

## 主な機能

### 📋 タスク管理
- 課題・出席コード・テストの期限を一括管理
- 期限切れ・今日・今後の予定に自動分類
- 相対時間表示（「3時間後」「2日後」）で締切を直感的に把握
- タスクの追加・編集・完了・削除
- 毎週繰り返し設定（週次の出席コードなどを自動作成）
- 登録済みの授業からタスクを紐づけ（手入力も可能）

### 🗓 授業管理
- 時間割を表形式で表示（月〜金 × 1〜6限）
- 授業の登録・編集・削除
- 今日の曜日列をハイライト表示
- 時間割のマス目タップで授業追加（曜日・時限が自動入力）

### ✅ 出席管理
- 出席・欠席・遅刻をワンタップで記録
- 欠席回数をプログレスバーで可視化
- 「あと○回休めます」の表示
- 欠席上限に近づくと赤色で警告

### 📅 カレンダー
- 月表示で各日にタスクのドットを表示
- 日付タップでその日のタスク一覧を確認
- 月全体でスケジュールを俯瞰

### 🎓 GPA・単位シミュレーション
- 現在のGPAを自動計算
- 「この科目を落としたら」のシミュレーション機能
- 取得済み単位数と卒業に必要な単位数の表示

### 🔔 Push通知
- 課題・テストの期限24時間前にブラウザ通知
- 出席コードの期限1時間前に通知
- PC・スマホ両対応

### その他
- ダークモード（システム設定連動 + 手動切替）
- PWA対応（スマホのホーム画面に追加してアプリのように使える）
- モバイルファーストUI（FAB、ボトムシート、ボトムナビ）
- レスポンシブデザイン
